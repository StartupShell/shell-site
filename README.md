## Local Set Up
Make sure you have the following installed

* bower
* sass

If not, install them

`$ [sudo] npm install bower -g`

`$ [sudo] gem install sass`

## Install dependencies
`$ [sudo] npm i && [sudo] bower i`

## Compiling & Running

*in separate sessions*

`$ npm run sass`

`$ npm run server`


## Anything on master get's auto deployed
